Import and Match Remittance Lines

The new concurrent program ‘Import and Match Remittance Lines’ imports the remittance lines into a new interface table and identifies the receipt for each remittance line. All the remittance lines matched with the receipts will be linked to the receipt and will be available for automatic and manual application to invoices. The output of the program will be as BI Publisher report. This report tell about the number of matched and unmatched remitted lines, number of imported remitted lines which are rejected as per the validation.

Automatic Cash Application Enhancement

“ Knap sacking “ The Smart Receipts program enhanced to knap sack using the remittance amount or the receipt amount. The process of knap sacking allowed the system to determine the candidate transactions which can be matched to the remittance lines not having reference values to match with in the lockbox remittance detail records using the Smart Receipts automatic matching rules. The Smart Receipts program also enhanced to knapsack using the receipt amount when there are no remittance lines in the lockbox detail records for the receipts and hence no reference values to use in matching to Transactions using the Smart Receipts automatic matching rules. This program will reduce the manual effort by 90% and users can import millions of transactions with minimal manual intervention.
